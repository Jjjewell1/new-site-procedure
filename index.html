🌐 Setup Environment

Install the tools you need before writing a single line of code.

⚙ Install Ollama

Ollama downloads and runs AI models locally on your machine.

# macOS / Linux — run this in Terminal
curl -fsSL https://ollama.com/install.sh | sh

# Windows — download from ollama.com/download

★ Tip

This is a one-time install. After that, you can pull models anytime.

💻 Pull a Model

ollama pull llama3.2

This downloads the model (~4 GB). It will be cached on disk and reused next time.

⚡ Install Cline for VS Code

# In VS Code, open the Extensions view (Ctrl+Shift+X) and search "Cline"

Install the extension. After install, you can start a chat with Cline to begin coding.
