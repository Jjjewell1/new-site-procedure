⚙ Start a Chat with Cline

In VS Code, go to the Cline panel (usually on the right side) and type your first message.

🎦 Describe What You Want

Cline reads your codebase, so be specific about what you need — a new feature, a bug fix, or a refactoring pass.

♽ Review the Plan

Cline will first outline his plan before touching any files. Read it carefully — if something looks off, ask for clarification.

✔ Give Feedback or Approve

You can say "yes", "go ahead", or just approve the plan. Cline will then start writing code step by step.

♻ Watch Progress in Real Time

Cline writes one file at a time and shows you what he's doing. If something goes wrong, he'll flag it and ask for your input.

⌫ Ask Questions Along the Way

Don't be shy — if you're unsure about something, just ask Cline. He'll answer and adapt his approach.

♾ When Done, Review the Result

Cline will report what he built and where to find it. Check the files yourself — run them, look at the output.

★ Tip

You can switch Cline modes: Chat (freeform), Task (goal-oriented), or Agent (autonomous). For this workflow, Chat is what you'll use most.
