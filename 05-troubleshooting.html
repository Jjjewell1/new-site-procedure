Issue: Site Not Loading — Check Cloudflare Tunnel Status

Symptom: You visit https://newsite.jewellcore.com and get a connection error or blank page.

  • Open the Cloudflare Dashboard → Zero Trust → Networking → Tunnels
  • Check that your tunnel is Running
  • If it shows as disconnected, restart the tunnel or check Coolify logs for errors

Issue: Push Rejected — Pull First

Symptom: git push origin main fails with a non-fast-forward error.

  • This usually means the remote has new commits you haven't pulled
  • Run 
    git fetch origin
    first, then 
    git pull origin main
  • If you intentionally want to force-push, use 
    git push --force-with-lease origin main
    (safer than plain --force)

Issue: Ollama Not Responding — Restart the Service

Symptom: Cline can't reach the local model endpoint.

  • Check if Ollama is running: open a terminal and run 
    ollama list
  • If nothing shows up, restart it with 
    ollama serve
  • Verify the model loads by running 
    ollama pull [model-name]
    if needed

Issue: Coolify Build Fails — Check Logs

Symptom: The deployment in Coolify reports a build error.

  • Go to the resource in Coolify → Logs tab
  • Look for specific errors (missing files, syntax issues, permission problems)
  • If it's a file copy issue, verify your build command matches the actual directory structure
